## 📌 **Installation & Setup**  

### 🔹 **Clone the repository**  
```sh
git clone https://github.com/Tusharxhub/XchangeXpert.git
```
### 🔹 **Navigate to the project folder**  
```sh
cd XchangeXpert
```
### 🔹 **Install dependencies**  
```sh
npm install
```
### 🔹 **Run the project**  
```sh
npm start
```
### 🔹 **Open in your browser**  
```
http://localhost:3000
```
